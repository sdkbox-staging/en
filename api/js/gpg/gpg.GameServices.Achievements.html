(static) Increment(params)

Increments an achievement by the given number of steps. Leaving the increment undefined causes its value to default to 1. The achievement must be an incremental achievement. Once an achievement reaches the maximum number of steps, it is unlocked automatically. Any further increments are ignored. This method has no callback.

(static) SetStepsAtLeast(params)

Set an achievement to have at least the given number of steps completed. Calling this method while the achievement already has more steps than the provided value is a no-op. Once the achievement reaches the maximum number of steps, the achievement is automatically unlocked, and any further mutation operations are ignored.
